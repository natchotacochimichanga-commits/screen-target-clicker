Features

Built for real automation sessions

Not just image matching — timing, zones, rules, and multiple instances working side by side.

Window targeting

Pick any visible window. Each app instance claims its own target so two copies never fight over the same window.

Image & rule matching

Add target images for instant clicks, or conditional rules where primary and subsection images must both match.

Scan zones

Crop where to search — a global zone for targets and separate purple zones per rule.

Live scan HUD

A compact preview pins to your target window with match markers, cooldown timer, and loop stats.

Timing controls

Global click cooldown, burst click limits, long pauses, and wait tracking so automation stays controlled.

Configs & themes

Setups auto-save while you work and reload on launch. Eight presets (Atlas, Neon, Ember, Forest, Rose, Slate, Ocean, Crimson), a color wheel for accent and panel colors, custom fonts, and exportable JSON configs.

In-app guide

The Help tab includes a full user guide — setup, targets, rules, webhooks, shortcuts, and multi-instance tips.

Discord webhooks

Send scan start/stop, errors, wait summaries, and optional click batches to your own Discord channel — no server join required.

Bugs & feedback

Report bugs, reviews, and recommendations from the app’s Feedback tab or on GitHub Issues — structured templates, no Discord needed.

Multi-instance

Run several copies at once — each with its own window, hotkey, and saved config file.

Hotkey & test mode

Toggle scanning with a global hotkey (F6 by default), preview detection before you start, and use keyboard shortcuts for save, load, and delete.

How it works

Up and running in minutes

  1. 1

    Download & extract

    Grab the portable zip from GitHub Releases, extract it anywhere, and run ScreenTargetClicker.exe.

  2. 2

    Choose a window & targets

    Select the app to watch, add screenshot targets, and optionally draw scan zones or conditional rules.

  3. 3

    Test, then start

    Use Test Detection to verify matches, tune confidence in Settings, then hit START or your hotkey.

  4. 4

    Close when done

    Your config saves automatically. Next launch picks up where you left off.

Failsafe: move the mouse to the top-left corner of the screen to abort PyAutoGUI actions.

Guide

Quick start

The full guide lives in the app under Help → Guide. Here’s the short version.

  1. 1

    Pick a window

    On the Window tab, select the app to watch. Use search if the list is long. Optionally draw a scan zone.

  2. 2

    Add targets or rules

    Targets are images to click instantly. Rules require a primary and subsection image to both match before clicking.

  3. 3

    Test, tune, start

    Run Test Detection, adjust confidence in Settings, then press START or your hotkey (F6 by default).

  4. 4

    Webhooks & alerts (optional)

    Under Help → Webhooks, paste a Discord webhook URL from your own channel to get scan alerts on your phone or desktop.
